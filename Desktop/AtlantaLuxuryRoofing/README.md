# Atlanta Luxury Roofing

Premium roofing lead generation site targeting Atlanta's luxury residential market.

## Site Structure

```
/                          → Homepage (index.html)
/standing-seam-metal-roof-cost-atlanta  → Core 30 Page #1
/natural-slate-roof-cost-atlanta        → Core 30 Page #2
/luxury-roofing-buckhead                → Neighborhood Page #1
```

## Deployment

Connected to Vercel. Every push to `main` branch auto-deploys.

**Domain:** AtlantaLuxuryRoofing.com
**Hosting:** Vercel
**Forms:** Netlify Forms / Formspree (update form handler in index.html)

## Adding New Pages

As new Core 30 pages are built, drop the `.html` file into this folder, commit, and push. Vercel deploys automatically. Update `sitemap.xml` with the new URL.
